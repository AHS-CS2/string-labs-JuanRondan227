# ReadMe Demo

This is the demo file for this project. It is being used to demonstrate the new work flow for all assingments.

- Clone repo OR git pull --no-edits
- Create/Move into branch for specific lab.
- Stage changes and commit
- When lab is finished, push to remote --> git push origin [name of branch]
- Go to GitHub and open a new Pull Request.
- Review autograder or feedback from Mr. Azam.
- When completed, Mr. Azam wil merge the Pull Request
